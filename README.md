# bounding
Normalized cross-browser `element.getClientBoundingRect`

```sh
npm install bounding --save
```

```js
var bounding = require('bounding')
```

## API
### `bounding(element)`
Get a normalized plain object representing <var>element</var>'s client bounding rectangle

```js
{
  bottom: 118.875,
  height: 97.4375,
  left: 19.5,
  right: 630.5,
  top: 21.4375,
  width: 611,
  x: 19.5,
  y: 21.4375
}
```
